WTT Team Match Formatter

WTT の団体戦結果を検索して、日本語書式でそのまま使える形に整える MVP です。 釜山 2024 のような通常形式と、ロンドン 2026 の Stage 1A / Stage 1B / Preliminary Round 想定のラウンド指定に対応しています。

London 2026 は大会前だと結果 API が空の可能性があります。その場合でも、round 指定は stage 1a / stage 1b / preliminary round を使える前提で作っています。

Translations JSON

選手名や国名の辞書をここで編集できます。下の一括追加欄では、`英語名 日本語名` を行ごとに貼り付けてまとめて反映できます。単一スペース区切りのときは最後の空白で分割します。

公開環境で `ADMIN_TOKEN` を設定した場合、この編集欄の読込と保存には Admin Token が必要です。
